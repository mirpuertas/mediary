# Navigation map

High-level navigation structure of the app:
```
Home
 ├─ Daily entry
 │   ├─ Sleep
 │   ├─ Mood
 │   └─ Medication intake
 ├─ Calendar
 │   └─ Daily entry (by date)
 ├─ Summary
 ├─ Medications
 │   ├─ Medication detail
 │   ├─ Medication groups
 │   └─ Group reminders
 └─ Settings

Quick flows:
- Notification → Quick intake
- Notification → Daily entry (sleep)
```
